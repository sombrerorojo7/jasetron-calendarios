# 📅 Jasetron - Sistema de Calendarios de Contenido

Sistema profesional para gestionar calendarios de publicaciones en redes sociales (Facebook e Instagram).

## 🚀 Acceso Rápido

**Editor Principal:** `panel_jasetron_HORIZONTAL.html`

## 📁 Estructura del Proyecto

```
jasetron-calendarios/
├── index.html                          # Página de inicio
├── panel_jasetron_HORIZONTAL.html      # Editor principal
├── proyectos/                          # Archivos .json guardados
└── calendarios_finales/                # Calendarios exportados para clientes
```

## 💡 Cómo Usar

### 1. Trabajar en un Calendario

1. Abre `panel_jasetron_HORIZONTAL.html`
2. Edita publicaciones (arrastra entre semanas, agrega nuevas, cambia imágenes)
3. Haz clic en **"💾 Guardar"** → descarga un archivo `.json`
4. Sube ese `.json` a la carpeta `proyectos/`

### 2. Continuar un Proyecto

1. Abre el editor
2. Clic en **"📂 Cargar"**
3. Selecciona tu archivo `.json` de la carpeta `proyectos/`
4. Continúa editando

### 3. Exportar Calendario Final

1. Cuando termines, clic en **"📥 Exportar calendario"**
2. Se descarga un archivo `.html`
3. Súbelo a `calendarios_finales/`
4. Comparte el link con tu cliente

## 🎯 Características

- ✅ **Drag & Drop horizontal** tipo Kanban
- ✅ **Guardar/Cargar proyectos** en formato JSON
- ✅ **Imágenes personalizadas** (carga desde tu computadora)
- ✅ **Vista previa** antes de exportar
- ✅ **Sin instalación** - funciona en el navegador
