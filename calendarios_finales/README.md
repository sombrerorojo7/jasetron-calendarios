# Calendarios Finales

Aquí se guardan los calendarios exportados (.html) listos para compartir con clientes.

## Cómo compartir:
1. Sube el archivo .html aquí
2. Ve al archivo en GitHub
3. Clic derecho → Copy link
4. Comparte ese link con tu cliente

El cliente podrá ver el calendario directamente en su navegador.
